Lossless and Lossy Audio

Lossless Audio Files

Examples of Lossless File Types

WAV, AIFF, FLAC, ALAC and Apple Lossless are all lossless audio files.

Why Use Lossless File Formats?

It is best to use lossless audio files if you have recorded your own audio and are keeping a file to archive. It is also good to rip audio to a lossless format so you have a high quality recording which you can then save copies in a lossy format if you choose.

Lossy Audio Files

Examples of Lossy File Types

MP3 is the most widely known and used lossy audio file. Other file types include AAC, Ogg Vorbis and WMA

How does lossy compression work?

Humans are not able to hear very high frequencies and very low frequencies. These are discarded in lossy formats. Some lossy formats discard very quiet sounds that the listener is unlikely to notice, especially in loud parts of a song.

Why Use Lossy File Formats?

Lossy formats such as MP3 are compatible with most players. Sound quality is still good so you will not notice any difference in quality unless you are in an ideal environment with top end equipment.
